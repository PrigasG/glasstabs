# Create a custom color theme for glassTabsUI

Create a custom color theme for glassTabsUI

## Usage

``` r
glass_tab_theme(
  tab_text = NULL,
  tab_active_text = NULL,
  halo_bg = NULL,
  halo_border = NULL,
  content_bg = NULL,
  content_border = NULL,
  card_bg = NULL,
  card_text = NULL
)
```

## Arguments

- tab_text:

  Inactive tab text color.

- tab_active_text:

  Active tab text color.

- halo_bg:

  Halo background.

- halo_border:

  Halo border.

- content_bg:

  Tab content background.

- content_border:

  Tab content border.

- card_bg:

  Inner card background.

- card_text:

  Inner card text color.

## Value

A named list of class `"glass_tab_theme"`.
